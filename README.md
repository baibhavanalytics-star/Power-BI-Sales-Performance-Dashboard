# Sales Performance Dashboard (Power BI)

This project presents an interactive **Power BI Sales Performance Dashboard** designed to analyze business performance using key metrics such as revenue, orders, average order value, and product sales.

The dashboard provides insights into **sales trends, category performance, customer behavior, and revenue contribution**, helping businesses make informed, data-driven decisions.

# Project Objective

The objective of this project is to analyze sales data and identify key business insights through interactive data visualization.

The dashboard focuses on:

- Monitoring important business KPIs
- Comparing month-over-month sales performance
- Identifying high-performing product categories
- Understanding customer purchasing patterns
- Identifying key revenue-generating products using Pareto analysis

---

# Key Performance Indicators (KPIs)

The dashboard tracks the following metrics:

- **Revenue**
- **Orders**
- **Average Order Value (AOV)**
- **Sold Products**

These KPIs help measure overall business performance and sales efficiency.

---

# Dashboard Features

- Interactive Power BI dashboard
- KPI cards for quick performance monitoring
- Month-over-month comparison
- Category performance analysis
- Customer behavior analysis
- Geographic customer distribution
- Pareto analysis for revenue contribution
- Dynamic filtering using slicers

---

# Key Insights

From the analysis, several important insights were identified:

• Revenue decreased from **₹22.44L in January to ₹20.67L in February** (~7.9% decline)

• Orders dropped from **945 to 867**, indicating lower transaction activity

• Average Order Value slightly increased, suggesting customers purchased higher-value items

• **Electronics remained the top-performing category**

• **Furniture showed the largest decline in sales**

• Pareto analysis revealed that **a small number of products contribute to the majority of total revenue**

---

# Business Recommendations

Based on the insights, the following strategies can help improve business performance:

• Focus marketing efforts on **high-performing products**

• Improve **conversion rate between viewed and purchased products**

• Investigate the decline in the **Furniture category**

• Use Pareto insights to prioritize **inventory management**

• Strengthen marketing in **high-demand customer regions**
